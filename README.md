Molecular Toxicity Prediction Model:
This project uses a Random Forest (Ensemble) model to predict whether a chemical compound is 'Toxic' or 'Non-Toxic' based on molecular descriptors.

Features
- EDA: Target distribution and feature correlation analysis.
- Preprocessing: Variance thresholding, correlation filtering, and standard scaling.
- Feature Selection: Identification of the top 15 predictive descriptors.
- Validation: 5-Fold Stratified Cross-Validation.
